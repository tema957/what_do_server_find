Концепция:
клиент осуществляет запрос на сервер  для получения набора прогнозируемых цен, на основе которых будет рассчитываться выделяемый на закупку бюджет
посредством загрузки на сервер exel-файла установленного формата.
через api. На выходе получаем видоизмененный exel-файл с добавленным столбцом "Цена".

Основная идея:

В дальнейшем хотим использовать LSTM

О модели:
- возможность подгружать данные о уже имеющихся ценах и заново запускать цикл для поддержания высокой точности предсказания
- использования алгоритмов бустинга CatBoostRegressor и XGBoostRegressor
- 

Используемые библиотеки:
numpy	 v. 1.22.0
pandas	 v. 1.4.1
openpyxl v. 3.0.9
scikit-learn v. 1.0.2
catboost
xgboost

1. Для успешной загрузки файлов формата exel в pandas dataframe необходимо скачать библиотеку openpyxl
2. При использовании OC Windows изменить расположение директории

Выводы:
Время не является ключывым фактором, влияющим на цену.
Наибольшей по модулю корреляцией обладают "Срок поставки" и "Плановый срок поставки", а также "Поставщик",
однако величина "Срок поставки" определяется уже после закупки, поэтому ключевыми величинами являются
"Плановый срок поставки" и "Поставщик".
